##使用python+pyh生成html的例子

###配置环境及运行
1.安装

- [pip](https://pypi.python.org/pypi/pip)
- virtualenv

    `pip install virtualenv`

- pyh
  - pyh无法从pip直接安装，请至[此处下载](https://github.com/hanxiaomax/pyh)并运行`python setup.py install`
    
2.创建虚拟环境并切换

```
virtualenv ENV
source ENV/bin/activate
```

3.运行demo

```
python demo.py
```
